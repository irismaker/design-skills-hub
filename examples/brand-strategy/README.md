# Brand Strategy Example

This example demonstrates using the branding skill to define comprehensive brand strategy and positioning.

## Scenario

**Company:** CloudSync
**Industry:** Cloud Storage / SaaS
**Stage:** Pre-launch, establishing brand foundation
**Goal:** Define brand purpose, values, positioning, and narrative

## Requirements

- Clear brand purpose and values
- Differentiated positioning vs. competitors (Dropbox, Google Drive)
- Compelling origin story
- Brand voice guidelines
- Messaging framework

## Solution

### Step 1: Use the branding skill

In Claude Code:
```
/branding

We're launching CloudSync, a cloud storage solution for creative professionals.
Help us define our complete brand strategy including:
- Brand purpose and core values
- Positioning against Dropbox and Google Drive
- Origin story
- Brand voice and tone
- Key messaging

Our differentiator: We focus on creative workflow integration (Adobe, Figma, etc.)
and offer unlimited versioning for design files.
```

### Step 2: Review Generated Strategy

The skill generates comprehensive brand strategy documentation:

#### Brand Purpose
"We exist to empower creative professionals to work fearlessly, knowing their creative assets are protected, organized, and always accessible."

#### Core Values
1. **Creative-First** - Built by creatives, for creatives
2. **Reliability** - Your work is too important to lose
3. **Simplicity** - Powerful features, zero complexity
4. **Collaboration** - Better work happens together
5. **Innovation** - Constantly evolving with creative workflows

#### Positioning Statement
"For creative professionals who need reliable cloud storage with creative workflow integration, CloudSync is a cloud storage platform that offers unlimited version history and native integration with design tools. Unlike Dropbox and Google Drive, which are built for general file storage, we understand creative workflows and protect every iteration of your work."

#### Differentiation Matrix

| Feature | CloudSync | Dropbox | Google Drive |
|---------|-----------|---------|--------------|
| Unlimited versioning | ✅ | ❌ (30 days) | ❌ (30 days) |
| Creative tool integration | ✅ Native | ⚠️ Limited | ⚠️ Limited |
| Visual file previews | ✅ Advanced | ✅ Basic | ✅ Basic |
| Collaboration tools | ✅ | ✅ | ✅ |
| Price point | $12/mo | $12/mo | $10/mo |

#### Origin Story
"CloudSync was born from frustration. As a design agency, we lost weeks of work when a client requested 'version 3' and we'd already overwritten it. We searched for a solution that understood creative workflows—unlimited versions, visual previews, seamless tool integration. When we couldn't find it, we built it. CloudSync is what we wished existed: cloud storage designed by creatives who've felt the pain of lost work."

#### Brand Voice & Tone

**Voice Attributes:**
- **Confident** - We know creative workflows
- **Supportive** - We've got your back
- **Clear** - No tech jargon, just solutions
- **Passionate** - We care about your work

**Tone Guidelines:**

| Situation | Tone | Example |
|-----------|------|---------|
| Product features | Confident, clear | "Every version, saved forever. Because 'final_final_v3' shouldn't be a thing." |
| Error messages | Supportive, helpful | "Upload paused. Check your connection and we'll pick up right where you left off." |
| Marketing | Passionate, creative | "Your best work deserves better than 30-day version history." |
| Support | Patient, empowering | "Here's how to set up Adobe integration in 3 steps." |

**Words We Use:**
- Creative professionals, designers, artists
- Work, projects, creations
- Protected, secure, reliable
- Seamless, integrated, native

**Words We Avoid:**
- Users, consumers (we say "creatives")
- Files, documents (we say "work", "projects")
- Cheap, affordable (we say "valuable", "worth it")
- Complicated, complex (we say "powerful", "capable")

#### Brand Archetype
**Primary:** The Caregiver
- Protective, supportive, reliable
- Nurtures creative work, prevents loss
- Always there when you need it

**Secondary:** The Creator
- Appreciates creativity, innovation
- Understands the creative process
- Enables better work

### Step 3: Apply to Marketing

```
Subheadline: "Unlimited version history and native creative tool integration.
              Because your best work deserves more than 30 days."
CTA: "Start Creating Fearlessly"
```

#### Social Media Voice
```
Tweet: "Remember when you needed 'final_v2' from last month and it was gone?
        Yeah, we fixed that. Unlimited versions. Forever.
        #CreativeWorkflow"
```
